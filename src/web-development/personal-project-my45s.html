Project: My45s

Project Info

This was created to help me while shopping for records. Sometimes I can't remember what I have/don't have. So I just wanted something the lists all the records and I can can filter by artist and song name.

I am currently creating a new project based on this. I'm going to expand it into a media library where I can search for all my records, books/comics, compact discs, cassettes, and maybe movies.
